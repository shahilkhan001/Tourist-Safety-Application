# Tourist Safety Application

Tourist Safety Application is a real-time safety platform designed to enhance tourist security through live GPS tracking, SOS emergency alerts, and instant communication features.

## Features

* Real-time GPS location tracking
* SOS emergency alert system
* Live communication using WebSockets
* Secure JWT-based authentication
* Admin-user interaction dashboard
* Interactive maps and location monitoring
* Responsive full-stack web application

## Tech Stack

### Frontend

* React.js
* CSS
* JavaScript

### Backend

* Node.js
* Express.js
* MongoDB
* Socket.io
* JWT Authentication

## Team Project

This project was developed collaboratively as a team project. I contributed to Login, Blockchain and authentication systems, real-time communication features, and deployment integration.

## Installation

1. Clone the repository
2. Install dependencies using `npm install`
3. Configure environment variables
4. Start frontend and backend servers

## Note

Environment variables, API keys, and sensitive configuration files are excluded from this repository for security purposes.
